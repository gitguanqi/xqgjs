# xqgjs

This is a like jquery package.

[View Chinese documents](./zh.md)

## Install

**Browser**:

import cdn

```html
<!-- Browser -->
<script src="https://xqgj.cc/xqcdn/libs/xqgjs/xqgjs.min.js"></script>
<!-- es module -->
<script type="module">
    import xqgjs from '../lib/xqgjs-esm.min.js';
</script>
```
