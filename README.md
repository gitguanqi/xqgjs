# xqgjs

This is a like jquery package.

## use

### import cdn

```html
<!-- cdn -->
<script src="https://unpkg.com/xqgjs/libs/gjs.min.js"></script>
```

### use npm

```sh
npm i xqgjs
```

### detail

```html
<div id="test"></div>
```

```js
let test = $g.dom('#test');
```
